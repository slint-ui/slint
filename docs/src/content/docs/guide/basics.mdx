### The *.slint file
Slint components are defined in a file with the `slint` extension such as `my-component.slint`.

The file is then structured as follows:

\{
  Imports

  Globals 

  Component A...

  Component B...
\}

<CodeSnippetMD imagePath="/src/assets/generated/test1.png" imageAlt='Example slint file'>
```slint
component MyButton inherits Text {
    color: black;
    // ...
}


export component MyApp inherits Window {
    preferred-width: 200px;
    preferred-height: 100px;
    Rectangle {
        width: 200px;
        height: 100px;
        background: green;
    }
    MyButton {
        x:0;y:0;
        text: "hello";
    }
    MyButton {
        y:0;
        x: 50px;
        text: "world";
    }
}
```

:::note[Note]
When developing Slint applications for Android or iOS, you can only use Rust as the programming language for now.
:::

While Slint is used in the same way for mobile and desktop applications, there are a few things that have to be kept in
mind during development. Mobile platforms usually feature an especially small screen and no hardware keyboard, which
have direct consequences that cannot be handled by the user interface framework, but are specific to the application.

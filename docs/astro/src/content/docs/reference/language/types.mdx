Every property and every expression has a type: `color`, `length`, or
`brush`. \{#sls.type.kinds}

<NotInSC>
The full Slint language has more types; this specification doesn't cover
them yet.
</NotInSC>

## `color`

A `color` is a color with red, green, blue, and alpha channels.
The default `color` is transparent. \{#sls.type.color}

## `length`

A `length` is a distance.
The default `length` is zero. \{#sls.type.length}

## `brush`

A `brush` defines the content that an element is filled with.
The default `brush` is transparent. \{#sls.type.brush}

## Conversions

An expression whose type isn't the type of the property only binds to it if
one of the conversions in this section applies. \{#sls.type.conversions}

A `color` converts to a `brush`:
the brush that fills with that single color. \{#sls.type.convert.color-to-brush}

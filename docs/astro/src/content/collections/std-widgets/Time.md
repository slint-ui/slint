`Time`

Defines a time with hours, minutes, and seconds.

-   **`hour`(int)**: The hour value (range from 0 to 23).
-   **`minute`(int)**: The minute value (range from 1 to 59).
-   **`second`(int)**: The second value (range form 1 to 59).

This section documents the elements, properties, callbacks, functions, structs
and enums that are part of the Slint SC safety-certified API surface.

Items in this reference are explicitly annotated as covered by Slint SC. Other
parts of the Slint language are documented in the [main Slint
documentation](https://slint.dev/docs/slint/) but are out of scope for safety
certification.

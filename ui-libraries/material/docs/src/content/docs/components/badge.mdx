A `Badge` is a small status indicator, often used to display a count or status on top of another component (such as an icon or button).

## Properties

### text
<SlintProperty propName="text" typeName="string">
The text or number displayed inside the badge. If empty, the badge is shown as a small dot.
</SlintProperty>

A `CheckBox` is a control that allows users to select one or more options from a set. It can be in three states: unchecked, checked, or partially checked (when `tristate` is enabled).

## Properties

### check-state
<SlintProperty propName="check-state" typeName="enum" enumName="CheckState" propertyVisibility="in-out">
The current state of the checkbox. Can be `unchecked`, `partially-checked`, or `checked`.

```slint "check-state: CheckState.checked;"
CheckBox {
    check-state: CheckState.checked;
}
```
</SlintProperty>


### enabled
<SlintProperty propName="enabled" typeName="bool" defaultValue="true">
Whether the checkbox is enabled and can be interacted with.
</SlintProperty>

### has-error
<SlintProperty propName="has-error" typeName="bool">
Whether the checkbox should display an error state.
</SlintProperty>

### tristate
<SlintProperty propName="tristate" typeName="bool">
Whether the checkbox supports three states (unchecked, partially checked, checked).
</SlintProperty>

## Callbacks

### checked-state-changed(check-state: CheckState)
Invoked when the checkbox state changes.

## Functions

### toggle()
Toggles the checkbox state. In tristate mode, cycles through unchecked → partially checked → checked → unchecked.

### set-check-state(check-state: CheckState)
Sets the checkbox state. If the state is `partially-checked`, the tristate property is set to true.

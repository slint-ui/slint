A `SegmentedButton` displays a group of related options as a single control, where only one option can be selected at a time.

## Properties

### current-index
<SlintProperty propName="current-index" typeName="int" propertyVisibility="in-out">
The index of the currently selected item.
</SlintProperty>

### items
<SlintProperty propName="items" typeName="[struct]" structName="SegmentedItem">
An array of segmented items, each containing an icon and text.
</SlintProperty>

## Callbacks

### index-changed(index: int)
Invoked when the current-index changes.

## Functions

### select(index: int)
Selects the item at the specified index.

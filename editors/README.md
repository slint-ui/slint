
# Editor Configuration for Slint

This folder contains extensions or configuration files for different editor to better support .slint files.
See [the Slint documentation](http://localhost:4321/docs/guide/tooling/manual-setup/) for details on how to configure your editor of choice.

## Editors

- [Visual Studio Code](https://docs.slint.dev/docs/guide/tooling/vscode/)
- [Kate](https://docs.slint.dev/docs/guide/tooling/kate/)
- [Qt Creator](https://docs.slint.dev/docs/guide/tooling/qt-creator/)
- [Helix](https://docs.slint.dev/docs/guide/tooling/helix/)
- [(Neo-)Vim](https://docs.slint.dev/docs/guide/tooling/neo-vim/)
- [Sublime Text](https://docs.slint.dev/docs/guide/tooling/sublime-text/)
- [JetBrains IDE](https://docs.slint.dev/docs/guide/tooling/jetbrains-ide/)
- [Zed](https://docs.slint.dev/docs/guide/tooling/zed/)
